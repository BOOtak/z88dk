.run [opts] filename [cmdline]

supported extensions:
exe nex o p sna snx tap z80

options:

-? = print dirs holding file
-p = print PATH env variable
-c = cd to dir containing file

-3 = load Nth filename in PATH
-r = cd to cwd on clean exit

exe options only:

-i = print text information
-l = list blocks in file
